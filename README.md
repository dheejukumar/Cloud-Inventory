# Cloud-Inventory-Management
I Have developed an Inventory Management System.A Serverless Inventory Management System bulit using AWS Services to help Buisness manage Products,Stock levels, and Record Sales and also Recieve Low Stock Alert.


Features:
1.Viewing and Inventory Product.
2.Add new Product.
3.Update stock level.
4.Low Stock Alert using SMS or SNS.
5.Record Sales Transcation.
6.Serverless and Scalable.
